# Análise de Mercado

Este repositório contém um exercício de análise de dados do mercado, usando Python e bibliotecas como pandas, matplotlib, seaborn e plotly.

## Objetivos

- Calcular média, mediana e desvio padrão do preço normal por categoria.
- Identificar categorias com maior variação nos preços.
- Plotar boxplot de preços para categorias com maior desvio padrão.
- Criar gráficos de barras e interativos da média de desconto por categoria e marca.

## Arquivos

- `analise.ipynb` - Notebook com todo o código.
- `mercado.csv` - Base de dados utilizada.
